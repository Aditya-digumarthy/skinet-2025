**Project Description**

Skinet is a full-stack e-commerce web application built to demonstrate the development of a real-world online store using modern technologies. The project showcases a professional-grade application developed with .NET for the backend API and Angular for the frontend client.

The application includes essential e-commerce features such as product browsing, filtering, user authentication, shopping cart management, order processing, and payment integration.

**Key Features**

.NET Backend API – Scalable and maintainable RESTful API built with the latest .NET technologies.

Angular Frontend – Responsive and dynamic single-page application that communicates with the backend API.

Product Catalog – Browse, filter, and sort products by category and brand.

Shopping Cart – Add, remove, and manage items with persistent cart functionality.

Authentication & Authorization – Secure user login and registration system.

Payment Integration – Stripe integration for handling secure checkout and payments.

Docker Support – Docker Compose configuration for running services such as SQL Server and Redis locally.
